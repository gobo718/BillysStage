# Billy Labs v3.1.0 — Run Analysis Foundation

## Added
- **RUN ANALYSIS** workflow inside the private Curator Discovery Engine.
- Reusable job status component with real job state, progress, completed, flagged, and failed counts.
- Pause and safe-cancel controls designed to scale to future bulk analysis.
- Selected-specimen preview and review report groups.
- Accept, Edit, and Ignore review actions; suggestions never overwrite curator data.
- Local deterministic analysis adapter so the complete workflow is testable before a remote vision provider is connected.

## Architectural boundary
The queue, status UI, and review lifecycle are provider-independent. Future AI vision can replace the analysis adapter without rebuilding bulk progress or review behavior.
