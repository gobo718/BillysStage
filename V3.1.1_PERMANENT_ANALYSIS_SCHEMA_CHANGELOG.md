# Billy Labs v3.1.1 — Permanent Analysis Schema

## Added
- Permanent IDs for analysis reports, suggestions, metadata fields, and controlled options.
- Canonical analysis field registry.
- `Dominant Color` stored as `field_color_dominant_general`.
- `Dominant Color Brightness` stored as `field_color_dominant_brightness`, with permanent options for Light, Medium, Dark, and Mixed.
- Four-part AI review presentation: Detected Characteristics, Visual Analysis, Additional Observations, and collapsed Remaining Categories.
- Review flags remain separate and never overwrite curator data.

## Deliberate boundary
The approved 15 general-color choices were not reconstructed from conversation memory. Their permanent option IDs will be entered from the canonical color notes when those notes are in hand.
